Attributes - Ատրիբուտներ, հատկանիշներ

HTML էլեմենտների հատկանիշները և հատկանիշներին վերագրվող արժեքները։

* (աստղանիշ)
ցանկացած

Global Attributes (Ընդհանուր հատկանիշներ)

Ընդհանուր հատկանիշները կարող են կիրառվել, գրեթե բոլոր HTML element-ների վրա։

Որոշ դեպքերում, չնայած֊որ՝ կարելի է կիրառել, սակայն՝ կարող է անիմաստ լինել։ Օրինակ՝ հենց "html tag"֊ի դեպքում, արգելված չէ կիրառել "id, class" attribute-ները, սակայն գործնականում, հազվադեպ հանդիպող երևույթ է։

Անվանում Բացատրություն
id իդենտիֆիկատոր, նույնականացուցիչ
class դասակարգիչ, դաս, խմբավորել, դասակարգել

Input element attributes (Մուտքագրման մասնիկի հատկանիշները)

Մուտքագրման դաշտի (<input>), type արժեքից կախված, կաորող է փոփոխվել տվյալ դաշտի հատկանիշները: Աղյուսակում ցուցադրված է, թե տվյալ հատկանիշը, որ տեսակների հետ է՝ կարող օգտագործվել։

Հատկանիշ (attribute) Տեսակ (type)
id *
class *
name (անուն) *
type (տեսակ, տիպ) *
value (արժեք) *
placeholder (տեղապահ) text, number, password
min (նվազագույն) number, range
max (առավելագույն) number, range
minlength (նվազագույն երկարություն` քանակ) text, password
maxlength (առավելագույն երկարություն` քանակ) text, password
